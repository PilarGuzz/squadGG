¡Bienvenidos a mi sitio web!

Este es un sitio web de ejemplo para demostrar cómo se puede construir una página web utilizando HTML y Bootstrap 5.

¿Qué es Bootstrap?

Bootstrap es un framework de diseño web que permite construir páginas web de manera rápida y eficiente. Bootstrap incluye una amplia variedad de componentes, como botones, formularios, tablas y mucho más.
